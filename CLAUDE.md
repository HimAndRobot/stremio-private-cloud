# Project Rules

- Never include `Co-Authored-By` in commit messages
- Do not add comments inside functions, only at the top of the function in English
- Language: all code, comments, and documentation in English
- Frontend: Vue 3 + Vite
- Backend: Node.js ESM (type: module), Express, SQLite via better-sqlite3
- Stremio addon uses IMDB IDs (prefix `tt`) with Cinemeta for metadata
- HTTPS via local-ip.medicmobile.org wildcard certificate for LAN access
